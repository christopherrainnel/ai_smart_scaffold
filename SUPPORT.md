# Support

For AI Smart Governance product questions, installation issues, account access, billing, bug reports, or feature requests:

- Email: `inquiry@hsnexus.com`
- Website: [hsnexus.com](https://www.hsnexus.com)
- Public issues: use this repository's GitHub issue templates when the report does not contain secrets, private code, customer data, payment details, or vulnerability details

## Account And Subscription Help

AI Smart Governance uses an H&S Nexus account for official governance access across devices.

- Free access is `$0/month` and is capped server-side at the first 2 governed repos per day plus up to 2 concurrent governed sessions/agents.
- Tier1 is `$3/month` and enables unlimited official-use repos and governed sessions/agents, subject to safety and abuse controls.
- Sign-in is mandatory for official governance. Signed-out use should not be treated as active governed product access.

If billing or subscription management is not yet exposed in the public dashboard, contact `inquiry@hsnexus.com` and include the email address tied to your H&S Nexus account.

## What To Include In A Public Bug Report

- Product surface: VS Code extension, plugin, workbench, CLI/MCP, website/account, or other
- Version or release channel, if visible
- Operating system and editor/tool version
- A short reproduction path
- Expected behavior and actual behavior
- Screenshots only if they do not reveal secrets, private code, private repo names, customer information, or payment data

Do not post secrets, Supabase URLs/keys, payment identifiers, private repository content, proprietary prompts, or diagnostic bundles in a public issue.

## Security Reports

Do not open a public issue for vulnerabilities. Follow [SECURITY.md](SECURITY.md) instead.
