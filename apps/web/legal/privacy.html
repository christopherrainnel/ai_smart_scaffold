Privacy Policy

How AI Smart Scaffold handles your data — local-first, privacy-respecting governed AI development.

Version 1.0 · Last updated April 1, 2026 · H&S Nexus Ventures

1. Who We Are

H&S Nexus Ventures ("HSNV", "we", "us", "our") develops and distributes AI Smart Scaffold, a governed runtime for AI-assisted software development, released first as a VS Code extension with associated local runtime components while connected product surfaces roll out in stages.

Contact: inquiry@hsnexus.com

2. Scope

This Privacy Policy applies to:

  • The AI Smart Scaffold VS Code extension
  • The AI Smart Scaffold CLI (python -m gov)
  • Any connected local AI Smart Scaffold product surfaces made available in this release line, including the Workbench desktop application where provided
  • Any optional HSNV-operated services you explicitly connect to through the Software (e.g., subscription or license APIs)

This Policy does not apply to your own project workspace files. Those remain entirely on your machine and are not subject to this Policy.

To keep Workbench and the VS Code extension aligned, the Software may store machine-local runtime state such as notification history, alert resolutions, timestamps, entitlement status, and workspace health markers on your device. That local state remains on your machine unless you deliberately include it in a diagnostic bundle or use an optional HSNV service.

3. Data We Collect

Data You Provide — When you purchase a subscription: name, email address, payment information (processed by our payment provider, not stored by HSNV), and license key details. When contacting support: content you choose to include in your request.

Optional Diagnostic Bundles — If you use the SEND BUGS / FEEDBACK feature, you may choose to submit a locally generated diagnostics bundle to HSNV over HTTPS. The bundle is generated on your machine for review before you confirm upload. Include only what is needed for troubleshooting. HSNV uses this material only for troubleshooting and analysis, and targets deletion once the support issue is resolved.

Automatically Collected — The Software operates primarily locally. The current runtime does not require telemetry to function. If optional telemetry is introduced, it is limited to anonymized feature activation events, error codes, extension version, and session counts.

We do not collect: Your source code, project files, file contents, commit messages, repository names, AI prompts, or model responses.

License Validation — If you use hosted license validation: license key hash (one-way), machine fingerprint hash, and subscription status are exchanged. No workspace content is transmitted. Validation requests are logged for up to 90 days.

8. Telemetry Controls

The current runtime does not require telemetry. If enabled, the product provides controls for enabling, disabling, or configuring it. In jurisdictions requiring prior consent (e.g., EU), telemetry is off by default.

9. Security

  • License key hashes stored using one-way SHA-256
  • Machine fingerprint data hashed before transmission
  • All communications use TLS 1.2 or higher
  • Scaffold content IP encrypted with AES-256-GCM at rest

10. Children's Privacy

AI Smart Scaffold is a developer tool intended for professional and adult use. HSNV does not knowingly collect personal data from individuals under 18.

11. International Data Transfers
