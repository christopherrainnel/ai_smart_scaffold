# AI Smart Scaffold 0.1.0

Version `0.1.0` is the first public release line for AI Smart Scaffold.

## Release Focus

This release establishes the initial public product surface for:

- installation guidance
- system requirements
- supported-tool guidance
- legal and security documents
- release notes and asset references
- the first-install Tier1 introductory workflow

## Included Product Forms

- VS Code extension
- Workbench desktop app
- packaged governed runtime components

## Notes

AI Smart Scaffold is designed as a local-first product. Repository-specific work remains local unless support material is deliberately shared by the user.

For `0.1.0`, the product applies the Free baseline immediately and then upgrades eligible first installations to Tier1 in the same workspace after a short local entitlement check. The bundled Tier1 window lasts 20 days and then returns to Free unless a separate paid Tier1 entitlement is active.
