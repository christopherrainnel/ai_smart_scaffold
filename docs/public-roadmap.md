# Public Roadmap

This roadmap is intentionally conservative. It describes public-facing direction without exposing private implementation details.

## Near Term

- Align public naming around AI Smart Governance.
- Keep this repository focused on docs, support, updates, security reporting, and issue intake.
- Prepare account-required onboarding copy for the official product.
- Finalize backend account, entitlement, quota, and billing behavior before broad release.

## Later

- Consider opening selected safe policy formats, examples, SDKs, or compatibility layers for inspection.
- Publish clearer install guidance when the official VS Code, plugin, or workbench product forms are ready.
- Expand public docs based on real user questions and support patterns.

## Not Promised

The main governance brain and official entitlement enforcement are not planned for automatic public release. Any future code-opening decision depends on owner approval, monetization risk, security review, and product maturity.
