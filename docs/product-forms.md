# Product Forms

AI Smart Governance may be distributed through several official product surfaces. The surfaces should all point back to the same signed-in account and entitlement truth.

## Official Surfaces

- VS Code extension
- Plugin integrations where supported
- Workbench app or local operator surface
- CLI and MCP runtime integrations
- H&S Nexus account, subscription, and entitlement services

## Public Repo Role

This repository documents the product and collects public reports. It does not ship the private governance brain or the official runtime source.

If selected compatibility layers or examples are opened later, they should be added deliberately with owner approval and clear boundaries so public code cannot be confused with the monetized official product.
