# System Requirements

## Runtime Core

| Requirement | Minimum |
|---|---|
| Python | 3.10+ |
| Recommended Python | 3.12 |
| Operating system | Windows 10+, macOS 12+, Linux with glibc 2.31+ |
| Disk space | At least 50 MB for runtime plus dependencies |
| Network | Not required for the local-first runtime path |

## VS Code Extension

| Requirement | Minimum |
|---|---|
| VS Code | 1.85+ |
| Python | 3.10+ for full runtime-backed operation |
| Node.js | Not required separately for normal use |

## Workbench

| Requirement | Current public review baseline |
|---|---|
| Desktop surface | Tauri-based Workbench app |
| Current prepared installer assets | Windows x64 `.msi` and setup `.exe` |
| Recommended use | Default onboarding and MCP connection management |

## Practical Notes

- Workbench is the preferred operator surface for setup.
- VS Code is the main in-editor surface for visibility and review.
- The product is local-first. Optional hosted commercial or entitlement services are separate from the core runtime path.
